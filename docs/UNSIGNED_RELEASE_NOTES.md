# LoopForge 1.0.0 Early Tester Build

**Unsigned early tester build for Apple-silicon Macs running macOS 13 or
newer.**

LoopForge is not signed with an Apple Developer ID. macOS will display an
unidentified developer warning.

## Install

1. Download the ZIP or DMG attached to this release.
2. Move LoopForge to Applications.
3. In Applications, right-click LoopForge and choose **Open**.
4. If macOS still blocks it, open **System Settings > Privacy & Security**,
   find the LoopForge security message, and click **Open Anyway**.

Only download this build from the official LoopForge GitHub repository.

## Downloads

- `LoopForge-1.0.0-arm64-unsigned.dmg`: drag-and-drop installer.
- `LoopForge-1.0.0-arm64-unsigned.zip`: direct application archive.
- `SHA256SUMS`: checksums for both app downloads and the compliance archive.
- `LoopForge-1.0.0-FFmpeg-Sources-and-Licenses.zip`: bundled FFmpeg/ffprobe,
  exact source archives, build scripts, checksums, and license notices.

Users must provide their own OpenAI/OpenRouter API key or run Ollama locally.
LoopForge does not provide a hosted AI service.
